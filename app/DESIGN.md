# 抽奖助手设计规范

## 品牌与色值
- 主色（Accent）：`#32F08C`（来源：TRAE 官网 CSS 变量 `--bg-bg-brand` 的 fallback）
- 背景：`#0B0B0B`
- 文本：`#E6E6E6`
- 次级文本：`#9AA0A6`
- 边框：`#1F1F1F`

## 字体与字号
- 字体：Inter / 系统字体族
- 标题：24px/700；正文：14–16px/400；行高：1.5

## 组件规范
- 按钮：圆角 12px；默认深色背景；Hover 边框与光晕使用主色；过渡 150–300ms。
- 卡片：深色背景、1px 边框；圆角 12px；分区间距 16px。

## 动画
- 中奖卡片：`pop` 入场（300ms）、`glow` 脉冲（1600ms 循环）。

## 可访问性
- 点击目标 ≥ 44px；暗色背景与文本对比度达到可读标准；键盘可聚焦。
